  • CRFs: Conditional Random Fields

  • EF: energy flux

  • F0: fundamental frequency

  • HMMs: Hidden Markov models

  • GMMs: Gaussian mixture models

  • DBN: Dynamic Bayesian network

  • DNNs:: Deep Neural Network (WHICH ONE????)

  • ODF: onset detection function
